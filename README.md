# FAQ
Domande frequenti per Monero project

## D: Che cosa è Monero?
R: Monero è una criptomoneta basata sul protocollo p2p e su una blockchain pubblica in cui vengono tracciate tutte le transazioni. A differenza di altre criptomonete come bitcoin o ethereum, Monero è l'unica ad avere un algoritmo sicuro e privato utilizzato di default per nascondere la tracciabilità delle transazioni. In poche parole nessuno può vedere quanti Monero hai o quanti ne hai spesi per quella transazione.
